# 对手模拟方法

## 何时读取本文件

当争议解决任务需要站在对方律师角度推演抗辩策略、质证意见或反诉可能性时，读取本文件。

## 本文件影响什么决策

- 决定报告中"对方抗辩模拟"章节的内容。
- 决定我方策略方案中需要提前防范的攻击点。
- 决定风险评估中的主要风险来源。

## 不适用场景

- 不用于评估我方证据的证明力（见 `evidence-assessment.md`）。
- 不用于确定法律依据（见 `legal-reasoning-methods.md`）。
- 不用于诉讼策略方案的整体设计（见 `dispute-resolution-methods.md`）。

## 1. 对方抗辩框架

站在对方律师角度，从以下维度推演可能的抗辩理由。

### 1.1 程序性抗辩（先看这些，一旦成立直接驳回）

| 抗辩类型 | 适用情形 | 判断要点 |
|---|---|---|
| 诉讼时效 | 对方权利可能已过时效 | 普通 3 年，起算点、中断/中止事由 |
| 管辖异议 | 受诉法院可能没有管辖权 | 合同约定管辖、被告住所地、合同履行地 |
| 主体不适格 | 原告不是真正的权利人 | 债权转让是否通知、代理权限 |
| 仲裁条款 | 合同约定了仲裁而非诉讼 | 仲裁条款是否有效 |
| 重复起诉 | 同一案件已经处理过 | 前后案件当事人、标的、请求是否相同 |

### 1.2 实体性抗辩

| 抗辩类型 | 适用情形 | 典型理由 |
|---|---|---|
| 合同不成立/无效 | 合同存在效力瑕疵 | 违反强制性规定、欺诈胁迫、恶意串通 |
| 合同可撤销 | 存在可撤销事由 | 重大误解、显失公平 |
| 履行抗辩 | 我方可能未完全履约 | 同时履行抗辩、先履行抗辩、不安抗辩 |
| 不可抗力/情势变更 | 客观情况导致无法履约 | 疫情、政策变化、自然灾害 |
| 损失不成立/过高 | 质疑损失的真实性和合理性 | 未实际发生、计算依据不足、可预见性规则 |
| 抵销 | 对方也欠我方钱 | 到期债权、同种类 |
| 减损义务 | 对方没有采取合理措施减少损失 | 放任损失扩大 |

### 1.3 推演方法

对我方案情中的每个关键事实，问自己：
- 对方最可能从哪个角度攻击？
- 对方能拿出什么反面证据？
- 如果我是对方律师，我最想攻击哪个要件？

## 2. 对方质证意见框架

对方律师拿到我方证据后，会怎么质证？用以下框架逐份推演。

### 四性质疑（逐份证据）

**真实性质疑**：
- "该证据为复印件，我方无法确认与原件一致。"
- "微信聊天记录未经公证保全，无法确认完整性。"
- "该签名与当事人已知签名不一致，申请鉴定。"

**合法性质疑**：
- "该录音系未经同意的秘密录制，取证方式违法。"
- "该证据涉及商业秘密，来源不明。"

**关联性质疑**：
- "该证据涉及的是另一笔交易，与本案无关。"
- "该邮件往来发生时间早于合同签订，不能证明合同履行情况。"

**证明力质疑**：
- "该证据为单方制作，缺乏对方确认。"
- "证人与原告有利害关系，证言可信度低。"
- "该证据为孤证，无法单独证明待证事实。"

### 策略性评价

对每份证据，判断：
- 对方可能重点攻击哪份证据？为什么？
- 对方质证时会不会"借力打力"（把我方证据变成对方论据）？
- 我方有没有提前防范的口径？

## 3. 反事实推理

通过构建"如果某个关键事实不存在"的假想场景，判断该事实对案件结果的影响。

### 操作步骤

1. **确定目标事实**：选择对案件结果影响最大的 1-2 个关键事实。
2. **构建反事实世界**：假设这个事实不存在或相反（例如"如果没有这份合同""如果对方确实按时交货了"）。
3. **推演法律结果**：在这个假想场景下，法律结论会怎样变化？
4. **比较差异**：有这个事实和没有这个事实，结论差在哪？
5. **评估脆弱性**：如果这个事实被对方推翻，我方还剩多少支撑？

### 使用场景

- 评估我方案件对某个关键证据的依赖程度。
- 预判对方攻击某个关键事实后，我方案件还站不站得住。
- 帮助确定哪些事实是"必须守住的底线"，哪些是"锦上添花"。

## 4. 类比推理

参照类似案件中对方的抗辩策略，推演本案对方可能怎么做。

### 操作步骤

1. **找类似案件**：通过案例检索找到法律关系和争议焦点相似的案件。
2. **看对方（败诉方）做了什么**：败诉方提出了哪些抗辩？法院怎么回应的？
3. **评估本案**：这些抗辩在本案中是否同样适用？
4. **预判效果**：哪些抗辩在类似案件中被法院采纳了？哪些被驳回了？

### 注意事项

- 类比只是参考，不是照搬。每个案件的事实不同，对方律师的策略也不同。
- 重点关注法院对各种抗辩理由的裁判态度，这比对方律师的策略更有参考价值。
- 如果类似案件中对方的某个抗辩被法院采纳，我方需要提前准备应对。

## 5. 反诉风险评估

评估对方是否可能提起反诉。

### 评估要点

1. **有没有反诉请求权基础**？对方对我方是否也有到期债权？
2. **反诉与本诉有没有牵连关系**？反诉请求是否基于同一合同或同一交易？
3. **反诉的时效是否还有效**？
4. **反诉如果成立，对我方的诉讼请求有什么影响**？（抵销、减少赔偿额、独立赔偿）

### 输出

在报告中用简短段落说明：
- 是否存在反诉风险（高/中/低/无）。
- 如果存在，对方最可能的反诉请求是什么。
- 反诉如果成立，对我方的影响有多大。

## 6. 冲突解决

当模拟过程中发现法条竞合或证据矛盾时，按以下规则处理。

### 法条竞合

如果同一个事实可能适用多条法条：
- 特别法优于一般法。
- 新法优于旧法。
- 上位法优于下位法。
- 无法判断时，列出所有可能的法条及其适用条件。

### 证据矛盾

如果不同证据之间存在矛盾：
- 分别列明矛盾内容。
- 比较每份证据的三性和证明力。
- 如果矛盾无法解决，标注为"需进一步核实"或"需律师判断"。

---

## 7. 能力边界

- 对手模拟是基于案情的合理推演，不是精确预测。对方律师的实际策略可能与模拟不同。
- 反事实推理是思维工具，结论具有假设性质，不应作为确定性结论。
- 类比推理受案件事实差异影响，仅作参考。
- 模拟结果应标注"对方可能的抗辩方向，需律师结合实际情况调整"。
